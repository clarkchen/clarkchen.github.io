Hexo启动教程

发表于 2020-11-11

首次启动

  • 首先把 项目 clone 到本地
  • 安装nodejs，建议你直接官网下载最新的编译器，NodeJS

  • 检查下版本， 以下为最新版本

  • 安装 hexo , 官网

  • 安装好之后，执行 hexo server 就能跑起来了

常见操作

1. 写一篇新的

官方教程，在命令行执行

  • hexo new post 炒菜-技术教程

如果你不是一气呵成的话，可以先新建一片草稿（不会在页面上显示）

  • hexo new draft 炒菜-技术教程-草稿版
